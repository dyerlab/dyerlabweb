Spatial Data Sources

Thu Oct 06, 2016 by Rodney J. Dyer in  Code, R, Research, spatial

As part of a class in Landscape Genetics, faculty (mostly done by Melanie Murphy and Jeffrey Evans) compiled an extensive list of spatial data sources. These were made available on the course website we hosted but I wanted to make a more persistent copy of them here so they will not be lost.
