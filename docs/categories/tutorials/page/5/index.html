Restriction Enzymes for AFLP’s

Fri Feb 01, 2013 by Rodney J. Dyer in  aflp, laboratory, protocols, Tutorials

The key to the AFLP protocol is to be able to digest two restriction enzymes (RE’s) simultaneously and be able to ligate onto these sticky ends primers of known concentration. When you purchase new primers, you need to aliquot out usable volumes because repeated freeze/thaw cycles reduce RE efficiency.
